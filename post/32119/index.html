字符串类型

字符串的内置操作

首先字符串内置的操作有**，::索引取值、切片、strip去除空格、split拆分、长度Len、成员运算in和not in::**这些。

索引取值：Python为所有的由多个元素组成按顺序排列的，都设置了索引的概念，因此我们可以通过索引取到对应的某个值，当然也可以反向索引

name = '张xx'
print(name[0]) # 张
print(name[-1]) # x

切片：可以将我们的字符串按照索引将区域内的值取出来，生成新的字符串，原有字符串不发生改变。

info = '好好学习，天天向上。'
print(info[0:4]) # 好好学习

注意 “习” 的索引号应该是3，但是Python有个“顾头不顾尾”这一个说法，因此我们要将索引+1

info = '好好学习，天天向上。'
# 第二个索引号不写，表示从第一个索引号（0）开始全部取完
print(info[0:]) # 好好学习，天天向上。
# 省去第一个索引号不写，表示从0开始取到停止
print(info[:4]) # 好好学习
info = '好好学习，天天向上。'
print(info[0:4:2]) # 好学

切片还可以指定步长，索引值0，2（由于“顾头不顾尾”的说法）因此结果是“好学”

当然还有反向步长，如果我们将步长值设为-1，第一个和第二个索引都不设置，就可以得到一个反向字符串了

info = '好好学习，天天向上。'
print(info[::-1])  # 。上向天天，习学好好

strip去除空格：不提供任何参数的时候，去除字符串左右两边的空格，提供一个字符串作为参数的时候，可以将字符串左右两边的字符去除了

info = '。。/好好学习，天天向上。/'
print(info.strip('。/'))  # 好好学习，天天向上

split拆分：当不提供任何参数的时候，默认使用空格进行拆分，当然也可以指定分隔符进行拆分，同时也可以指定拆分次数。返回结果为列表。

info = '好好 学习 天天 向上'
print(info.split())  # ['好好', '学习', '天天', '向上']
info = '好好，学习，天天，向上'
print(info.split('，'))  # ['好好', '学习', '天天', '向上']
# 指定拆分次数，即当拆分到1个的时候，就不再拆分
print(info.split('，', 1))  # ['好好', '学习，天天，向上']

长度len ：不仅可以计算字符串的字符个数，还可以计算列表，字典的元素个数

info = '好好 学习 天天 向上'
print(len(info), type(len(info)))  # 11 <class 'int'>

成员运算：可以判断某个字符串是否在某个字符串内

info = '好好 学习 天天 向上'
print('好好' in info)  # True

其他操作：

info = '=/A-B-c-D/='

# .lstrip()和.rstrip()
# 和strip()一致，只是l代表left r代表right
print(info.lstrip('=/'))  # A-B-c-D/=
print(info.rstrip('=/'))  # =/A-B-c-D

# .rsplit() 从右向左进行分割
print(info.split('-'))  # ['=/A', 'B', 'c', 'D/=']
print(info.rsplit('-'))  # ['=/A', 'B', 'c', 'D/=']
# 当然 从左往右 和 从右往左 全部分割完肯定没区别
# 当我们指定第二个参数的时候就有区别了
print(info.split('-', 1))  # ['=/A', 'B-c-D/=']
print(info.rsplit('-', 1))  # ['=/A-B-c', 'D/=']

# .lower()和upper()
# 将所有大写字母变成小写字母 将所有小写字母变成大写字母
print(info.lower())  # =/a-b-c-d/=
print(info.upper())  # =/A-B-C-D/=

# join()
# 将一个列表拼接为一个字符串，也可以指定字符串，当然这个列表里的值都必须是字符串
l = ['你', '好', '世', '界']
print(' '.join(l))  # 你 好 世 界

# replace()
# 将字符串内某些指定字符串替换掉，并生成一个新的字符串
l = '我好喜欢你，别分开好么？'
print(l.replace('好', '好不'))  # 我好不喜欢你，别分开好不么？
# 当然也可以指定替换的 次数
print(l.replace('好', '好不', 1))  # 我好不喜欢你，别分开好么？

列表类型

首先我们来了解一下，如何生成一个列表，你可能会疑惑，这个问题我们在第一章已经解决了。还记得我们第一章是怎么解决的么？

hobbies = ['打羽毛球','可乐','读书']

我们直接给了hobbies一个列表，那这个列表是怎么被造出来的，其实这是list()的功能，还记得类型转换么？list()就是可以进行类型转换的，那么它可以把什么类型进行转换呢？？？🧐

答案是任何可迭代对象，list()可以把任何可迭代对象转换成list类型。

列表的内置操作

首先，列表的内置操作有，::索引取值、切片、追加、插入、extend、长度、成员运算、删除、计数::

我们可以把字符串类比为一个列表，因此列表的很多内置操作和字符串都是大同小异的。上面加粗的是我们会讲的，其他类比字符串类型即可。

索引取值：首先，字符串也可以索引取值，但是为什么我们还要在列表中也讲一遍，因为列表还可以 索引改值。因为列表是可变类型，字符串是不可变类型。就像下面一样。

l = ['李慧','高启','赵李']
l[1] = '高启强' * 2
print(l)  # ['李慧', '高启强高启强', '赵李']
# 复习了一下字符串的操作，不知道还记不记得那个*的意思

追加：如果我们想要在索引3的位置增加一个“高纬”，那我们就要用到l.append(内容)

l = ['李慧','高启','赵李']
l.append('高纬')
print(l)  # ['李慧', '高启', '赵李', '高纬']

插入：如果我们要想在列表在第1号索引的位置，插入一个值，其他值往后移动。就要用到insert(索引值，内容）

l = ['李慧','高启','赵李']
l.insert(1,'高纬')
print(l)  # ['李慧', '高纬', '高启', '赵李']

extend：如果我们有另外的一个列表我们想把两个列表合并，我们该使用extend(可迭代对象）

l = ['李慧','高启','赵李']
l2 = ['赵高']
l.extend(l2)
print(l)  # ['李慧', '高启', '赵李', '赵高']

删除：我们之前学过用del来解除一个变量和一个值的关系，现在如果我们要删除列表内的某个元素。就要用pop(索引值)【不传值默认删除最后一个】remove(元素)：可以删除指定的元素。

l = ['李慧', '高启', '赵李']
del l[0]
print(l)  # ['高启', '赵李']

l = ['李慧','高启','赵李']
l.pop(1)
print(l)  # ['李慧', '赵李']

l = ['李慧','高启','赵李']
l.remove('李慧')
print(l)  # ['高启', '赵李']

其他操作：

# count(元素) 查找某个元素出现了几次
l = ['李慧', '高启', '赵李', '李慧']
print(l.count('李慧'))  # 2
# index(元素) 查找某个元素的索引号，从左往右找，找到就停
print(l.index('李慧'))  # 0
# clear() 清空某个列表
l.clear()
print(l)  # []
# reverse() 反转一个列表
l = ['李慧', '高启', '赵李', '李慧']
l.reverse()
print(l)  # ['李慧', '赵李', '高启', '李慧']

排序：我们可以用sort()实现对列表的升序排序，如果传入reserve = True就可以实现降序排序

l = ['a','z','d','m','c','s']
l.sort()
print(l)  # ['a', 'c', 'd', 'm', 's', 'z']
l.sort(reverse=True)
print(l)  # ['z', 's', 'm', 'd', 'c', 'a']

元组类型

这是我们要学习的一个新的数据类型，元组就是 不可变的列表

tuples = ('李慧', '张明', '康熙', '鹿海')
three = ('李慧',)
# 由于小括号的特殊性，导致元组只有一个元素的时候，就要加个逗号。
# 元组（tuple）与列表类似，不同之处在于元组的元素不能修改。元组写在小括号 () 里，元素之间用逗号隔开。
print(tuples)  # ('李慧', '张明', '康熙', '鹿海')
# 可以把字符串看作一种特殊的元组
print(tuples + three)  # ('李慧', '张明', '康熙', '鹿海', '李慧')
# 元组的元素不可改变，但它可以包含可变的对象，比如list列表
tuplist = ('李慧', '张明', '康熙', '鹿海',['张自成','李自健'])
print(tuplist)  # ('李慧', '张明', '康熙', '鹿海', ['张自成', '李自健'])
# 下面的语句是对 tuplist第3索引 的列表进行操作
tuplist[4][1] = '张自民'
print(tuplist)  # ('李慧', '张明', '康熙', '鹿海', ['张自成', '张自民'])

那么既然可以包含可变的对象，那么元组到底不可变了什么呢？

不可变的是存的内存地址，还记得列表在内存的存储方式么

它不可变的，就是值“a”对应的内存地址等这些。

元组和列表

元组是用来存储一组不变的值，那如果我们定义一个列表，我们不去改它，就让他在那里，不就好了么？元组存在的意义是什么呢？列表可以改也可以写。如果一组数据，只有读取的需求，而没有写的需求，那么当然是元组更加节省内存空间，当然，也没有那么省，所以，最好还是该使用元组的时候使用元组。

元组的内置操作

由于元组的不可修改性质，我们元组有的操作只有**::索引取值、切片、统计长度、成员运算、循环以及index和count::**等。

字典类型

我们之前说过字典类型的key只可以是字符串类型和数字类型，那么这两种类型都统称为“不可变类型”，那元组也可以作为字典的key。

造一个字典/产生一个字典

如何造一个字典呢？我们一开始在基本数据类型中使用的是{key : value, key : value}这样的格式来造出一个字典的，那这个字典是怎么被造出来的，其实这是 dict() 的功能。下面我们来介绍几种其他造字典的方法。

# 第一种 用dict来造一个字典
dic = dict({'name':'李慧','home':'浙江'})
print(dic)  # {'name': '李慧', 'home': '浙江'}

# 第二种 dict(变量名=值,变量名=值)的格式来创建字典
dic = dict(name='李慧',home='浙江')
print(dic)  # {'name': '李慧', 'home': '浙江'}

# 第三种formkeys 如果我们要初始化一个字典，将所有值设为None
keys = ['name','home']
dic = {}.fromkeys(keys, None)
print(dic)  # {'name': None, 'home': None}

字典的类型转换

字典的类型转换有点不太一样，由于其特殊的key:value的性质，其传入的可迭代对象，必须是要成对出现的。

info = [['name','李慧'],['home','浙江']]

字典的基本操作

列表取值/改值：这个和列表差不多，只是把索引变成了key。如果::字典的key不存在，是可以根据key来添加值的。::字典没有插入和增加值（insert和append）这是因为字典是无序的，列表是有序的。

dic = {}
dic['name'] = '王慧'
print(dic)  # {'name': '王慧'}

更安全的取值：我们之前学过字典的取值，是使用[key]来取值的，但是当我们取一个根本不存在的key，Python就会给我们报错了。get(key)的方法就是更安全的取值。

info = [['name','李慧'],['home','浙江']]
dic = dict(info)
print(dic.get('a')) # None

就像上面这样。

删除值：首先通用的删除方法del，这个和列表的差不多。然后就是pop，clear，popitem

dic = {'name': '李慧', 'home': '浙江'}
del dic['name']
print(dic)  # {'home': '浙江'}

dic = {'name': '李慧', 'home': '浙江'}
print(dic.pop('name')) # 李慧
print(dic)  # {'home': '浙江'}

dic = {'name': '李慧', 'home': '浙江'}
print(dic.popitem())  # ('home', '浙江')
print(dic) # {'name': '李慧'}

注意一下，popitem返回的是一个元组，它清除的字典的最后一个值（字典本身是无序的）这个最后，是根据其放入的先后顺序得到的，也就是最后放入的值，会被先删除。

字典的内置方法：keys() values() items() 迭代器

dic = {'name': '李慧', 'home': '浙江'}
print(dic.keys())  # dict_keys(['name', 'home'])
print(dic.values())  # dict_values(['李慧', '浙江'])
print(dic.items())  # dict_items([('name', '李慧'), ('home', '浙江')])

集合类型

这是一个新的数据类型，它的作用并不是用来取值和存值的，它是用来做去重和关系运算的。集合可以存入任何不可变类型，那就说明，列表就是不能存入的。集合是不可以重复的，就算你定义的时候，写了多个但是还是只会存一个。集合是不能取值的，其本身是作为一个整体使用的，如果你使用索引去取值，就会报错。如果要取值就可以将集合转成list类型。

可以使用大括号 { } 或者 set() 函数创建集合，::注意：创建一个空集合必须用 set() 而不是 { }，因为 { } 是用来创建一个空字典。::

关系运算

a = set('abracadabra')
b = set('alacazam')
print(a)  # {'a', 'r', 'c', 'b', 'd'}
print(b)  # {'a', 'm', 'z', 'l', 'c'}
print(a - b)     # a 和 b 的差集
print(a | b)     # a 和 b 的并集
print(a & b)     # a 和 b 的交集
print(a ^ b)     # a 和 b 中不同时存在的元素
